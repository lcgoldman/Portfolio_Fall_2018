American Race Horses

Why?

Because I love racehorses! Back in the day I used to own racehorses. Standardbreds. Their sheer beauty and athleticism is something to behold. I wanted to share my passion and give an introduction to the breeds of horses that are used in horseracing in America as well as some of the similarities and differences between them.

Skills Used:

  • HTML5
  • CSS3

Outcome:

This was the first project I completed in the TechWorks program. In this class we were introduced to HTML5 and CSS3. The project was intended to demonstrate all the syntax that we learned in class. The website doesn't look all that polished because we had to use elements that might not have necessarily been appropriate or needed for a website such as this.

I do like how the animated racehorses came out in the overlay. With a little more practice and expertise I can improve upon that animiation. When I do, I will update this page.

Bed & Breakfast Reservation System (Simulation) New!

Why?

I love Bed & Breakfast's, that's why! And secondly, because it gave me a chance to practice using Javascript. And lastly, because it was a school assignment!

Skills Used:

  • HTML5
  • CSS3
  • Javascript
  • jQuery

Outcome:

This was the first project I completed for the TechWorks program Javascript class. In this class we were introduced to Javascript and jQuery. The project was intended to demonstrate all the syntax that we learned in class. I really like how this Project came out. I think the website gives the look and feel of a Bed & Breakfast, and would encourage users to want to make a reservation!

The program is still incomplete. There's still some functionality that needs to be added, and once I learn MYSQL, I will be able to make this a fully functional reservation system. When I make this website "complete", I will update this page, and then look out Expedia!!
